## рҹҢҹ What is SAS?

**Sentra Academic Solutions (SAS)** is Indonesia's **first multi-agent AI platform** purpose-built for Health Sciences academia. While generic AI tools answer questions, SAS orchestrates **7 specialized intelligent agents** that guide students, lecturers, researchers, and healthcare professionals through the *entire* academic writing lifecycle вҖ” from formulating a research question to walking into the thesis defense room with confidence.

Developed by **Sentra Healthcare Artificial Intelligence** in partnership with **Institut Ilmu Kesehatan (IIK)**, SAS is deeply rooted in Indonesian academic standards, local institutional requirements, and the real challenges faced by health sciences students every day.

```
ChatGPT writes.   Paperpal edits.   Jenni AI drafts.
SAS does all of it вҖ” and takes you all the way to graduation.
```

## рҹҡҖ Quick Start

### Prerequisites
- Active **Claude.ai** account (Pro or Team plan recommended)
- Access to **Claude.ai Projects** feature

### Setup in 4 Steps

**Step 1 вҖ” Create a Project**
```
claude.ai вҶ’ Projects вҶ’ Create Project
Name: "Sentra Academic Solutions"
```

**Step 2 вҖ” Add Custom Instructions**
```
Project Settings вҶ’ Set custom instructions
вҶ’ Paste the full contents of: sentra-academic-solutions-project-instructions.md
```

**Step 3 вҖ” Upload Knowledge Base** *(Optional but Highly Recommended)*

| Document Type | Examples |
|---------------|----------|
| рҹ“ҳ Writing Guidelines | Thesis manual, citation format guide, chapter templates |
| рҹҸӣпёҸ Institutional Templates | Cover page, approval sheet, originality declaration |
| рҹ§Ә Methodology Guides | Research SOPs, ethical clearance forms |
| вҡ–пёҸ Regulations | Rector's decrees, ministerial academic standards |
| рҹҸҶ Sample Theses | 2вҖ“3 exemplary theses as quality benchmarks |
| рҹ“Ҡ Assessment Rubrics | Defense rubric, manuscript eligibility checklist |
| рҹ“ү Statistical Guides | Test selection flowcharts, interpretation tables |

**Step 4 вҖ” Start Chatting!**
```
Hi SAS, I'm a 7th-semester Nursing student.
I want to research the relationship between sleep quality
and academic performance. Where should I start?
```
